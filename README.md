# IELTS Writing Coach

An AI-powered IELTS Academic Writing platform built to help students achieve Band 8 and 9.

## Features
- AI Chat with IELTS Writing Coach
- Essay Evaluator with band scoring
- Question Generator (Cambridge style)
- Daily Practice exercises
- Vocabulary builder
- Grammar lessons
- Linkers library
- Templates with smart finder

## Tech Stack
- Next.js 14, TypeScript, Tailwind CSS
- Clerk Authentication
- Supabase PostgreSQL
- Groq AI (Llama 3.3)

## Setup
1. Clone the repo
2. Copy `.env.example` to `.env.local` and fill in your keys
3. Run `npm install`
4. Run `npm run dev`
